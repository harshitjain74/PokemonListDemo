# pokemontcg: Basic Pokémon Card List

**Note:** The code and demo must be shared via mail:


# Features:

- Create a Pokémon card list and search using the Api’s from the below
   https://domain/cards?page=1&pageSize=10
   https://domaino/cards?q=query
- App bar with search text box
- List cards with Pagination (image and title with 2X2 grid)
- On click card open detail page with card details like. Attack, set, type, weakness, artist and
   many more ( with animation )
- Manage App-Route


# Tech Stack to be used:
1. Display image with Caching
2. Use State-Management
